# Gitmojis

Use these emojis to represent the commit type visually:

| Emoji | Code | Description |
|---|---|---|
| ✨ | `:sparkles:` | Introducing new features. (feat) |
| 🐛 | `:bug:` | Fixing a bug. (fix) |
| 📝 | `:memo:` | Writing docs. (docs) |
| 💄 | `:lipstick:` | Updating the UI and style files. (style) |
| ♻️ | `:recycle:` | Refactoring code. (refactor) |
| ⚡️ | `:zap:` | Improving performance. (perf) |
| ✅ | `:white_check_mark:` | Adding tests. (test) |
| 👷 | `:construction_worker:` | Adding CI build system. (ci) |
| 🔧 | `:wrench:` | Changing configuration files. (chore) |
| 🔨 | `:hammer:` | Heavy refactoring or build scripts. (build) |
| ⏪ | `:rewind:` | Reverting changes. (revert) |
| 🔀 | `:twisted_rightwards_arrows:` | Merging branches. |
| 🚀 | `:rocket:` | Deploying stuff. |
| 🎉 | `:tada:` | Initial commit. |

Refer to [gitmoji.dev](https://gitmoji.dev/) for a complete list if needed.
